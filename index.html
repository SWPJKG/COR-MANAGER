⚙️ Google Sheets Setup

📋 One-Time Setup (~5 min) — Never expires!
  1. Go to sheets.google.com → create a new blank spreadsheet → copy its Sheet ID from the URL:
    …/spreadsheets/d/SHEET_ID/edit
  2. Go to console.cloud.google.com → create a project → enable the Google Sheets API
  3. APIs & Services → Credentials → + Create Credentials → Service Account → give it any name → Create
  4. Click your new service account → Keys tab → Add Key → Create new key → JSON → a .json file downloads
  5. Open the .json file in Notepad → copy all the text → paste it into the box below
  6. Find the client_email in the JSON → open your Google Sheet → Share → paste that email → give it Editor access
✅ This credential never expires — set it once and forget it.
